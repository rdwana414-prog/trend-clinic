# 🏥 Trend Clinic

Modern Healthcare Clinic Website - Bilingual (Arabic & English)

## Features

✨ **Stunning Design**
- Modern, responsive UI with animations
- Fully bilingual (Arabic & English)
- Mobile-friendly

📅 **Advanced Booking System**
- Two-step appointment booking
- Doctor selection
- Date & time picker
- Confirmation page

🎨 **Technology Stack**
- Next.js 14
- React 18
- Framer Motion (animations)
- Tailwind CSS (styling)
- i18next (multilingual)

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment

Deployed on Vercel - Visit: https://trend-clinic.vercel.app
